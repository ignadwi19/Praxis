Command prompt berbeda dengan powershell.
powershell dapat berguna untuk menulis script agar lebih mudah saat mengelola sistem Windows ketimbang Command Prompt.
powershell dapat menggunakan command yang berbeda yg biasa dikenal dengan cmdlets yang dapat dimanfaatkan dari mengubah registry
sampai mengatur windows management instrumentation (WMI) dimana comand prompt tidak bisa melakukan fungsi tersebut.
singkatnya adalah, powershell lebih kompleks dan lebih powerfull dibandingkan dengan comand prompt.

Untuk bekerja di dalam team dibutuhkan sebuah version control. salah satu version control adalah GIT. GIT bisa berkolaborasi 
dengan GIT HUB untuk membentuk suatu sistem yang mendukung team untuk bekerja bersama (dalam hal ini adalah programmer).
GIT memiliki banyak fungsi, yang paling utama adalah dapat melakukan tracking tentang aktivitas yang dilakukan melalui commit.
ketika sebuah file diubah maka GIT dapat menyimpan file baru tersebut melalui commit yang dilakukan. apabila file ingin kembali
kepada kondisi semula maka dapat kembali ke commit-commit sebelumnya. 
dengan menggunakan GIT HUB programmer bisa bekerja secara modular. selanjutnya dapat dilakukan merge di dalam remote (GITHUB) yaitu
untuk menggabungkan pekerjan-pekerjaan yang selesai dibuat.
